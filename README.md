# Upcoming Music

Search for a musical artist and see if they have any upcoming tour dates.

## Setup

#### Install

```sh
$ npm install
```

## Run
```sh
$ npm start
```

## Deploy
```sh
$ npm run deploy
```
This command will generate a production build of the app and deploy it to GitHub Pages.
